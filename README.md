# Aura: The Modern Wellness Alarm Clock

Aura is a beautifully crafted mobile alarm clock designed to elevate your mornings and optimize your day. Built with Flutter, Aura blends smart wake technology, intuitive time management, and calming design into a seamless crossâ€‘platform experience. Go beyond basic alarms with intelligent wake windows, engaging alarm challenges, analog/digital clock displays, and tools that help you focus and unwind.

## Features
- Elegant Interfaces: Clean analog/digital clock, alarm list, timer, and stopwatch.
- Smart Wake (Premium): Gently wakes you during a light sleep window for a refreshed start.
- Alarm Challenges: Optional puzzles or tasks to prevent accidental snoozing/dismissal.
- Ambient Soundscapes: Curated nature sounds and premium tones for a calm wake-up.
- Focus Timer: Built-in Pomodoro-style timer to structure deep work sessions.
- Cross-Platform: Flutter-powered experience across iOS and Android.

## Getting Started
1. Install and verify Flutter:
   - flutter --version
2. Fetch dependencies:
   - flutter pub get
3. Run the app:
   - flutter run

To list available devices:
- flutter devices

## Build
- Android (APK):
  - flutter build apk --release
- iOS (requires macOS and Xcode):
  - flutter build ios --release
